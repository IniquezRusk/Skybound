Skybound is a 2D side-scrolling arcade shooter built in C++ using SDL2.
The project focuses on fast-paced movement, smooth rendering, and arcade-style gameplay inspired by classic scrolling flight games.

Players control a plane traveling through a side-scrolling world while avoiding obstacles, navigating enemy threats, and surviving increasingly intense sections. The game is being developed as a personal project to strengthen skills in game programming, object-oriented design, real-time input handling, rendering, and gameplay systems.

Features
Side-scrolling 2D gameplay
Player-controlled aircraft movement
Real-time keyboard input
SDL2-based rendering pipeline
Arcade-style game structure
Modular C++ project organization
Foundation for enemies, collision, scoring, and level progression
